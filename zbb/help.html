Zebrafish Brain Browser Project

The Zebrafish Brain Browser (ZBB) enables visualization of the 3D expression patterns of transgenic and enhancer trap lines in zebrafish, developed by the Burgess lab at the Eunice Kennedy Shriver National Institute for Child Health and Human Development, in collaboration with the Advanced Research Computing group at Virginia Tech.

The atlas includes brain images of transgenic lines scanned at high resolution and registered to a reference brain so that all lines can be compared in the same spatial framework. For more information, please see: https://science.nichd.nih.gov/confluence/display/burgess/Brain+Browser

There are two interfaces for ZBB. You are currently looking at the online version (zbbrowser.com). You can download a desktop version from the link above.

The online version runs in any web browser. Javascript must be not be disabled. If accessing ZBB from an older computer or laptop, you may need to select 'L' (low) resolution from the Settings Tab and deselect the 'Show Volume' option. You will probably not be able to use Virtual Reality mode, and performance will lag if you try to view more than 2 lines at once.

If you'd like to cite ZBB, please use: Tabor KM et al. Brain-wide cellular resolution imaging of Cre transgenic zebrafish lines for functional circuit-mapping. Elife. 2019 Feb 8;8:e42687 https://elifesciences.org/articles/42687

Slice and Projection Views

On startup, the top left, top right and bottom left windows display a single sagittal, coronal and horizontal slice through the brain of a HuC:Cer transgenic larva.

The scroll bars allow you to change the position of the slice. The position (in microns) in the ZBB reference brain is indicated at the bottom left corner of each image. The blue cross hairs in each image indicate the position of the corresponding slices in the other two images.

Left-click on any point in any of the three images to change the position of the slices displayed in the other two images. The anatomical identity (if known) at the selected point is displayed beside the 'Region' label below the 3D window. To outline the boundaries of the region that includes the selected point use shift-left-click.

Zoom into an area using the mouse scroll wheel. Hold down the shift-ctrl and left-mouse button to move the position of the zoomed image.

To expand the image for any plane, select the blue arrow icon in the top right of each image. Return to the multiple-slice view by clicking on the same icon

The top-left corner of the sagittal (top left) image provides a color-coded indication of the lines currently selected.

To see a maximum projection through any plane, select the 'Max' button in the top-right of each image. Return to a single slice view by clicking 'Slice'. You can also generate partial projections through selected brain subregions by right-clicking twice in one image to designate the boundaries of the partial projection. A yellow line indicates the boundaries. For example, if you right-click slightly to the left and right of the olfactory placodes in the horizontal (bottom-right) plane, the coronal view (top-right) will now show a coronal projection of the olfactory bulbs. Clear a projection and return to single-slice view by selecting 'Slice'.

Summary of mouse controls:
Left button: change selected slices
Shift-left button: display anatomical region around selected point
Scroll wheel: zoom
Control-shift-left button: pan zoomed image
Right button: set boundaries for partial projections

Keyboard shortcuts:
a: show all anatomical regions
f: find text match
m: maximum projections
x: toggle cross-hairs
z: reset to 1x zoom
1-9: show/hide selected line

3D Projection

The bottom-right image is a 3D rendering of the brain volume. Use the mouse to rotate and zoom the 3D image:

Left button: free-rotation of the 3D perspective
Right button: zoom
Control left button: pan image

Clicking on the axis-arrows at the bottom-right of the image start and stop rotation of the image in each direction.

To reset the 3D projection to its default position, click on the grey ball in the middle of the axis- arrows.

Expand the 3D view using the arrow icon at the top-right of the image.

Click on the VR icon to open a new browser window for a virtual-reality 3D view of the brain. The virtual reality view is designed to work either with Google Cardboard or using a wide-view side-by-side viewer to produce a stereo image from the images on the monitor.

For Cardboard you need to access the website using your cellphone, select the lines you are interested in, then slip the phone into a cardboard viewer. Depressing the cardboard button moves you forward in whatever direction you are currently looking. You will need a high-end phone and may still be limited in the number of lines that you can have simultaneously displayed.

For side-by-side viewing you need a wide-view stereo viewer to produce a stereo view from the two images on your computer monitor. We use catalog #PVN-SWV from 3dstereo.com (also available from Amazon). Left mouse button rotates. Scroll wheel to zoom. Shift-left to pan.

You can toggle visibility of lines using the checkboxes at the top of the window.

Lines Tab

ZBB contains image stacks for around 250 transgene expression patterns. Patterns are organized into four tabs:
- Transgenic: transgenic lines using BAC recombination or defined promoters to drive a reporter
- Gal4: enhancer traps expressing Gal4
- Cre: enhancer traps expresssing Cre
- Misc: everything else (immunohistochemistry, dye fills, various other enhancer traps)

Each tab contains a list of lines with a checkbox, the name of the line, and a reference for the line (generally where it was first published). Clicking on the reference opens a new browser tab with the corresponding PubMed entry.

Clicking on the checkbox for a line downloads its image stack and adds it to the brain view. Use the control panel at the bottom of the tab to change the display color and intensity.

Finally, the 'Custom' tab allows you to:
(1) Load your own data into the browser.
(2) Generate a new pattern that is the predicted intersect of currently selected patterns.

Control Panel

The control panel allows you to adjust the display for each currently selected line, and access further information.

The name of the line is displayed in a button on the left. Click on the button to reveal a color palette and select a new display color. Click on the button again to close the palette. The bottom-right element in the palette is the ImageJ Fire look-up table.

Brightness/contrast are adjusted using the sliders or by directly entering values in the fields to the right of the sliders.

On the right side of the line beside the name of the line are 5 additional icons. From right to left:
- X: removes this line from the display
- Reset: returns brightness/contrast/color to defaults.
- i: opens a pop-up window with more information about the line including links to PubMed records for its generation and confocal scanning, and ZFIN entry (click-through to ZIRC to order lines). Anatomical annotation and genomic integration site is also provided if available.
- eye icon: toggle visibility of line
- number (1-9): keyboard shortcut for toggling visibility

Anatomy Tab

Show boundaries of either manually annotated neuroanatomical regions (Z-Brain, Randlett et al Nature Methods 2015) or computationally derived regions (Pajevic, Gupta et al Methods 2018).

First, select either the 'Z-Brain' or 'Pajevic' button at the top of the tab. Then use checkboxes to toggle display of different regions. The Z-Brain regions are organized by larger brain areas.

The Reset button turns off all selected regions.

Aliases Tab

Quickly find cell-types or brain regions of interest that are well-labeled by specific transgenic lines. These are simply shortcuts to finding specific transgenic lines that label structures of interest. In most cases a selected line also labels other structures, and may not label all of the selected structure.

Selecting a checkbox simply loads the corresponding transgenic line. Its display qualities can then be adjusted using the control panel.

Search Tab

The Brain Browser provides two kinds of search: spatial or text. Use the buttons at the top of the tab to chose spatial or text.

Text search: Enter free text in the 'search' field. As you type, all information (including anatomical information, integration site, publications history etc) associated with a line is searched and matches displayed in a table below.

Spatial search: Search for lines that show expression in a selected 3-dimensional volume.
Steps:
(1) Click the orange 'Start' button.
(2) Hold down the left-mouse button to select an area in any of the three slice windows.
(3) Hold down the left mouse button in a different slice window to complete the volume

The selected volume is indicated as a yellow box visible in the slice windows. Lines that express reporter in the selected volume are displayed in the table below, with the line name and the percentage of the volume selected that is labeled by that line.

Settings Tab

If ZBB is laggy on your computer, then (1) Turn off 'Show Volume' and (2) Select Low resolution.

Invert color - flips color-scales so that the brain is grey on a white-background (and all other colors are simply inverted as well).

Show volume - toggles display of the bottom-right 3D view of the brain. Deselect this on less powerful computers to improve performance.

Show cross-hairs - toggles visibility of the cross-hairs that indicate the plane of section for each slice view.

Hide annotations - select to remove all annotations in the slice windows - useful if you are taking a screenshot.

Projection depth coding - color-codes lines by depth when in a projection view (no effect in slice view).

Resolution - select either Low, Medium or High image quality. Selecting Low will improve rendering performance by your computer, and download speed from our servers.

Window Size - to best fit your monitor, select either Small, Medium or Large to adjust the relative size of the image slices and tab windows

Download Tab

Options to download a stack with all lines that are selected, or the current set of slices.

Download the currently selected stack as a set of 419 horizontal images. The png images are in a compressed folder that you can unzip. If multiple lines are selected, the stack will simply be the composite of these lines with the existing colors as currently displayed on your screen.

Note, image quality will be inferior to the original data, which is also available:
Gal4 lines (confocal scans/averages): http://gigadb.org/dataset/100322
Cre lines (confocal scans/averages): https://datadryad.org/resource/doi:10.5061/dryad.tk467n8
Everything (masked averages): https://zenodo.org/record/2560759

Share Web Link

Looking at an interesting region in a specific transgenic line? This Tab lets you generate a code so that you can share your current view with somebody else, including the set of lines you have selected and position in the brain.
Steps:
(1) Click 'Generate' to create a URL that encodes the current view
(2) Click 'Copy' to copy the URL to your clipboard
(3) Paste the URL into an email to share with a colleague

The shareable URL does not include changes you have made to the resolution or screen size.
